## 🧠 What This App Does

merge-and-rebase is built for model work in a few common cases:

- Merge two or more model checkpoints
- Rebase task vectors for better model alignment
- Fine-tune models for a new task
- Work with vision models and LLMs
- Use Transformer-based model files
- Handle OpenCLIP-style model workflows

In plain terms, the app helps you combine model changes and save the result as a new model file.
